# 効果音

ゲーム内で使用する効果音の一覧。デフォルトON、タイトル画面左下と案内画面でトグル切替可能。設定は localStorage に永続化。

## 素材元

[効果音ラボ](https://soundeffect-lab.info/) のフリー素材。商用無料・報告不要。

## 実装

- 再生基盤: `src/shared/utils/sound.ts`
  - `preloadSounds()` — 起動時に全効果音をプリロード（App.tsx で呼び出し）
  - `playSound(name)` — 効果音を再生（ミュート時は無音）
  - `isSoundEnabled()` / `setSoundEnabled(value)` — ON/OFF の取得・設定
- ミュートUI: タイトル画面左下の🔊/🔇アイコン + 案内画面の「効果音: ON/OFF」ボタン
- 音量調整: `sound.ts` 内の `VOLUME` 定数で効果音ごとに設定可能
- ファイル配置: `public/data/sounds/`
