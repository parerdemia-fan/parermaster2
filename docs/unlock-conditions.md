# アチーブメント・解放条件

バッジ、称号、隠し要素の獲得条件と解放の仕組みをまとめる。

---

## 1. バッジ

### 獲得条件

- **全問正解**が必須
- **顔名前当て**: 全問題タイプ（顔当て/名前当て/名前を作ろう）が出題される
- **知識クイズ**: 問題数30問（不足時は全問）で全問正解

### 進化段階

全問正解で獲得。難易度に応じて同じ画像が進化（上位で上書き）。

| レベル | 条件 | 見た目 |
|--------|------|--------|
| ブロンズ | ふつうで全問正解 | 小さい・控えめ |
| シルバー | むずかしいで全問正解 | 中サイズ・光沢 |
| ゴールド | 激ムズで全問正解 | 大きい・キラキラ |

## 2. 称号

| 称号 | 条件 | 備考 |
|------|------|------|
| 2期生マスター | 2期生エリア全スロットが最大レベル | 特別演出 |
| 1期生マスター | 1期生エリア全スロットが最大レベル | 特別演出 |
| パレ学マスター | 2期生マスター + 1期生マスター | シークレット（達成まで非表示） |
| パレ学グランドマスター | パレ学マスター + タイムアタック7分21秒以内クリア | パレ学マスターのアップグレード。シークレット |

※「最大レベル」= 2期生知識クイズはシルバー、他はゴールド

## 3. 解放条件

### 全体マップ

```
ふつうで全問正解 → ブロンズ獲得
  → むずかしいで全問正解 → シルバー獲得 → 【激ムズ解放】
    → 激ムズで全問正解 → ゴールド獲得
      → 世代マスター称号 → 【タイムアタック解放】
        → 両世代マスター → パレ学マスター → 【スタッフロール解放】
```

### 激ムズ（難易度3）

| 項目 | 内容 |
|------|------|
| 条件 | 該当スロットでシルバーバッジを獲得（= むずかしいで全問正解） |
| 粒度 | スロットごとに独立 |
| 通知 | 結果画面のバッジ獲得演出に含める（既存実装） |

### タイムアタック

| 項目 | 内容 |
|------|------|
| 条件 | **両方の世代を制覇**（世代別スロットが全て最大レベル） |
| 補足 | 2期生: 全員ゴールド + 知識シルバー。1期生: 全員ゴールド + 知識ゴールド |
| 補足 | 寮別エリアの4スロットは条件に含まない |
| 通知 | 結果画面でバッジ獲得演出の後に「タイムアタックが解放されました！」の特別演出 |
| 導線 | タイトル画面にボタン出現。初回は「NEW」ラベル付き（1回プレイで消える） |


### スタッフロール

| 項目 | 内容 |
|------|------|
| 条件 | **パレ学マスター称号を獲得**（両世代の全スロットが最大レベル） |
| 通知 | パレ学マスター称号の獲得演出に含める |
| 導線 | アチーブメント画面から（未定） |
