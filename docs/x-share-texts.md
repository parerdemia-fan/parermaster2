共通ハッシュタグ: `#パレ学マスター #パレ学`
共通URL: `https://parerdemia-fan.github.io/parermaster2/`

## 1. 通常ゲーム結果（ResultScreen）

**実装**: `src/app/routes/ResultScreen.tsx:71-78`

```
【パレ学マスター 2nd Season 結果発表】
{genLabel} {modeLabel} {diffLabel}
{パーフェクト時: 🎉🎉🎉パーフェクト達成！🎉🎉🎉 / それ以外: {correctCount}/{total}問正解（{rate}%）}
{messageLine1} {messageLine2}{badgeText}

👇挑戦はこちら
{GAME_URL}
#パレ学マスター #パレ学
```

例（通常）:
```
【パレ学マスター 2nd Season 結果発表】
1期生 顔名前当て ふつう
18/25問正解（72%）
👏 いい調子！ リスナーさん、なかなかの実力！

👇挑戦はこちら
https://parerdemia-fan.github.io/parermaster2/
#パレ学マスター #パレ学
```

例（パーフェクト＋バッジ）:
```
【パレ学マスター 2nd Season 結果発表】
1期生 顔名前当て ふつう
🎉🎉🎉パーフェクト達成！🎉🎉🎉
🎉🎉🎉 リスナーさん、パーフェクト達成！ 🎉🎉🎉
🏅 顔名前当て ゴールドバッジ 獲得！

👇挑戦はこちら
https://parerdemia-fan.github.io/parermaster2/
#パレ学マスター #パレ学
```

### 改善案

- **「結果発表」が硬い**。テレビ番組的で、カジュアルなゲームのシェアとしてはやや浮く。「やってみた」「遊んでみた」の方がXの文化に馴染む
- **パーフェクト時にメッセージが冗長**。「🎉🎉🎉パーフェクト達成！🎉🎉🎉」の直後に「🎉🎉🎉 リスナーさん、パーフェクト達成！ 🎉🎉🎉」と実質同じ内容が2行並ぶ。パーフェクト時はスコア行を省略してメッセージ行だけにするか、メッセージ行をシェアから外すことを検討
- **「👇挑戦はこちら」が宣伝感が強い**。シェアの主目的は「自分の結果を見せたい」であって「他人に遊ばせたい」ではない。URLだけ置くか、もっと自然な誘導（「あなたも挑戦してみて」等）にすると押しつけ感が減る
- **genLabel / modeLabel / diffLabel が3つ並ぶ**と情報量が多い。パレ学を知らない人には意味不明。ただしプレイヤーにとっては「何をクリアしたか」の証拠なので残す価値はある
- **ゲームURLがハッシュタグの前にある**ため、Xのプレビューカードが生成される。この配置は維持すべき

### 修正案

通常時:
```
パレ学マスター 2nd Season
{genLabel} {modeLabel} {diffLabel}
{correctCount}/{total}問正解（{rate}%）
{messageLine1} {messageLine2}

{GAME_URL}
#パレ学マスター #パレ学
```

パーフェクト＋バッジ:
```
パレ学マスター 2nd Season
{genLabel} {modeLabel} {diffLabel}
🎉 {playerName}さん、パーフェクト達成！
🏅 顔名前当て ゴールドバッジ 獲得！

{GAME_URL}
#パレ学マスター #パレ学
```

変更点:
- 【結果発表】→ ゲーム名のみ（硬さを解消）
- パーフェクト時の重複行を解消（メッセージ行のみ、🎉も1つずつに簡潔化）
- 「👇挑戦はこちら」を削除、URLのみ（押しつけ感を軽減）

## 2. タイムアタック結果（TimeAttackResultScreen）

**実装**: `src/app/routes/TimeAttackResultScreen.tsx:41`

```
{playerName}: {timeMessage}

パレ学マスター 2nd Season タイムアタック
クリアタイム: {formatTime(clearTime)}

👇挑戦はこちら
{GAME_URL}
#パレ学マスター #パレ学
```

例:
```
リスナー: まだまだやれるはず！

パレ学マスター 2nd Season タイムアタック
クリアタイム: 1:23.45

👇挑戦はこちら
https://parerdemia-fan.github.io/parermaster2/
#パレ学マスター #パレ学
```

### 改善案

- **通常ゲームと構成が統一されていない**。通常は `【】` の墨付き括弧タイトルだが、タイムアタックにはない。フォーマットを揃えた方がブランドとして認識されやすい
- **タイムの数値が目立たない**。タイムアタックのシェアの主役は記録（タイム）なので、もっと大きく見せたい。例: `⏱ 1:23.45` のように絵文字で視覚的に強調
- **自己ベスト更新時の特別感がない**。実装にはisNewRecordフラグがあるが、シェア文言に反映されていない。「🎊 自己ベスト更新！」等を追加すると共有モチベーションが上がる

### 修正案

通常時:
```
パレ学マスター 2nd Season タイムアタック
⏱ {formatTime(clearTime)}
{playerName}: {timeMessage}

{GAME_URL}
#パレ学マスター #パレ学
```

自己ベスト更新時:
```
パレ学マスター 2nd Season タイムアタック
⏱ {formatTime(clearTime)} 🎊自己ベスト更新！
{playerName}: {timeMessage}

{GAME_URL}
#パレ学マスター #パレ学
```

変更点:
- 【】付きではないが通常ゲームとフォーマットを揃えた（ゲーム名が1行目）
- タイムを⏱絵文字付きで2行目に配置（主役を目立たせる）
- 自己ベスト更新時に🎊を追加（シェアモチベーション向上）
- 「👇挑戦はこちら」を削除

## 3. 相性診断結果（DiagnosisResultScreen）

**実装**: `src/app/routes/DiagnosisResultScreen.tsx:42-47`

```
相性診断の結果✨
相性の良い寮生は…
🥇 {name}（{dormLabel}）
🥈 {name}（{dormLabel}）
🥉 {name}（{dormLabel}）

{GAME_URL}
#パレ学マスター #パレ学
```

例:
```
相性診断の結果✨
相性の良い寮生は…
🥇 犬丸なでこ（バゥ寮）
🥈 花晴りら（ミュゥ寮）
🥉 飛渡ココ（クゥ寮）

https://parerdemia-fan.github.io/parermaster2/
#パレ学マスター #パレ学
```

### 改善案

- **ゲーム名が入っていない**。他のシェアには「パレ学マスター 2nd Season」が入っているが、診断にはない。Xで流れてきたときに何のゲームかわからない
- **相性度(%)が含まれていない**。画面上はパーセンテージが表示されるのにシェアには含まれない。「🥇 犬丸なでこ（バゥ寮）98%」のように入れると具体性が増し、会話のきっかけになる
- **「👇挑戦はこちら」がない**。他の画面との統一感もあるが、診断系は「あなたは誰と相性がいい？」のような問いかけの方がクリック率が高い（診断系シェアの定石）
- **プレイヤー名が含まれていない**。通常ゲームには含まれるが診断にはない。ただし診断は名前なしの方が自然かもしれない

### 修正案

```
パレ学マスター 2nd Season 相性診断✨
相性の良い寮生は…
🥇 {name}（{dormLabel}）{similarity}%
🥈 {name}（{dormLabel}）{similarity}%
🥉 {name}（{dormLabel}）{similarity}%

あなたは誰と相性がいい？
{GAME_URL}
#パレ学マスター #パレ学
```

変更点:
- 1行目にゲーム名を追加（何のゲームかわかるように）
- 相性度%を追加（会話のきっかけになる）
- CTAを「あなたは誰と相性がいい？」に（診断系シェアの定石、問いかけ形式）

## 横断的な改善案

### Xシェアの目的の整理

シェア文言は2つの目的を同時に果たす必要がある:

1. **シェアする人の動機**: 自分の結果を見せたい、自慢したい、面白い結果を共有したい
2. **見た人への効果**: 興味を持ってもらう、遊んでもらう、パレデミア学園を知ってもらう

現状は(2)の「遊ばせたい」に寄りすぎている（「挑戦はこちら」等）。(1)の「見せたい」を満たすことが先で、それが自然に(2)につながるのが理想。

### 具体的な横断改善

| 項目 | 現状 | 提案 |
|------|------|------|
| ハッシュタグ | `#パレ学マスター #パレ学` 全画面共通 | 統一されているのは良い。維持 |
| URL配置 | ハッシュタグの前 | OGPプレビュー生成のため維持 |
| CTA文言 | 「👇挑戦はこちら」「👇今すぐプレイ」が混在 | 結果シェアは控えめに、紹介シェアだけ強めにする |
| ゲーム名 | 診断結果のみ欠落 | 全シェアに統一的に含める |
| フォーマット | 通常は【】タイトル、他はなし | 最低限「パレ学マスター」のワードは全シェアに入れる |
