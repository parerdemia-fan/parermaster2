# UI画像アセット一覧

[メイン仕様書](specification.md) > UI画像アセット一覧

---

## 方針

- パネルは**単一画像**（CSS伸縮）— 9-patchは不採用
- ボタンは**3-patch**（left / middle / right）
- ラベルは全て**テキスト描画**（画像へのテキスト埋め込み廃止）
- バッジは共有画像 + CSSエフェクトで進化表現（前作の27枚 → 最大10枚）
- 画像配置先: `public/data/images/`
