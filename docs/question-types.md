## 1. 顔名前当ての進化

| 世代 | 方式 |
|------|------|
| 前々作 | 顔当て・名前当てが別々のゲームモード |
| 前作 | 「顔名前当て」として顔当て・名前当てがランダムに混合出題。知識クイズは別モード |
| **2nd** | 顔当て・名前当てに加え新しい出題パターンを追加。知識クイズモードの問題（テキストクイズ）もタイムアタックでは問題タイプの一つとして出題 |

- 各問題タイプごとに難易度を設ける（段階数は問題タイプにより異なる。必ずしも3段階でなくてよい）
- 問題タイプの追加が容易なプラグイン的アーキテクチャを採用する（詳細は [architecture.md](architecture.md)）
- **問題タイプは今後追加する可能性がある**

## 2. 出題方式

- 合計問題数は**出題範囲のタレント数**と同じ（例: 15名なら全15問）
- 有効な問題タイプで均等に分割し、問題タイプごとにまとめて出題する
  - 出題順は難易度の低い順: **顔当て → 名前当て → 名前を作ろう**
  - 例: 15名 → 顔当て5問 → 名前当て5問 → 名前を作ろう5問
- 各タレントは全体で1回だけ出題される（問題タイプをまたいで重複しない）
- 出題範囲の全タレントが出題対象（問題数の選択肢は不要）

### 問題難易度（★表示）

ゲーム画面のヘッダーに表示する問題の難易度。設定難易度（ふつう/むずかしい/激ムズ）と問題タイプの組み合わせで決まる。

| 設定難易度 | 顔当て | 名前当て | 名前を作ろう |
|-----------|--------|---------|-------------|
| ふつう（★☆☆） | ★1 | ★1.5 | ★2 |
| むずかしい（★★☆） | ★2.5 | ★3 | ★3.5 |
| 激ムズ（★★★） | ★4 | ★4.5 | ★5 |

- ★の最大値は7（タイムアタックで★6〜★7を使用）
- 半端な値（★1.5等）は塗り半分の星で表現する
- 知識クイズ（テキストクイズ）の問題難易度は問題データの difficulty 値に基づく（別途定義）

### 通常モードとタイムアタックの対応

- 問題タイプ1〜3（顔当て・名前当て・名前を作ろう）: 通常モードの顔名前当て + タイムアタック
- 問題タイプ4（テキストクイズ）: 通常モードでは「知識クイズ」モードとして独立出題。タイムアタックでは他の問題タイプと混合出題（出題範囲は世代を問わず全問題）
- 問題タイプ5〜7（名前はどこ？・スポットライト・ぼかし）: **タイムアタック専用**

### テキストクイズ（知識クイズモード）の補足

- 前作の「基本問題」（difficulty 0）と「深堀り問題」（difficulty 1〜5）を難易度で一本化
- 2期生の知識クイズについて:
  - 2期生固有の深堀り問題は十分な問題数が用意できるまで保留
  - ただし、パレ学や寮自体に関する問題（タレント個人に依存しない問題）は前作からそのまま流用可能
  - これにより2期生の知識クイズも初期リリースからある程度の問題数で提供できる

### テキストクイズの表示仕様

#### タレント名選択肢の判定ルール

- 生成時に全4選択肢がタレント名（talents.json に一致）かを判定
- 全一致: `answerTalentIds` にタレントID配列をセット → 2×2 グリッド表示
- 一部不一致: `answerTalentIds = null` → 通常の4行テキスト表示

#### 全選択肢がタレント名の場合（2×2 グリッド）

- 各セル: タレント顔写真 + 下部に名前ラベル
- 回答後: 正解セルは緑枠、不正解セルは赤枠、他は半透明

#### hideIcon の挙動

- `hideIcon = false`（デフォルト）: 回答前から常にタレントアイコン/顔写真を表示
- `hideIcon = true`:
  - **回答前**: 顔写真の代わりに 👤 プレースホルダー。名前は表示
  - **回答後**: プレースホルダーが実際の顔写真に切り替わる

#### 問題文・解説文のインラインタレントアイコン

- テキスト中のタレント名を検出し、名前の**後ろ**にインラインアイコンを挿入
- 検出: talents の name を長い順にソートして正規表現マッチ（部分一致防止）
- 問題文: hideIcon に従い showIcon を制御（hideIcon=true かつ未回答 → 👤）
- 解説文: 回答後のみ表示されるため、常に実画像を表示

#### 回答後の解説表示

- 解説文（comment）: タレント名のインラインアイコン付きで表示
- 解説画像（commentImage）: 解説文の上に表示
- 情報源リンク（sourceUrl）: 解説欄の末尾に「📎 情報源: サイト名」としてリンク表示（URLからサイト名を自動判定: YouTube, X, TikTok, パレデミア学園公式サイト等）

#### ナビゲーション（知識クイズモード限定）

- 回答後に「← 前へ」「次へ →」ボタンを表示
- 回答前でも「← 前へ」ボタンは表示（前の解説を見返せる）
- 戻った問題は閲覧専用（回答済み状態で表示、回答変更は不可）
- 顔名前当て等の他の問題タイプでは「前へ」ボタンは表示しない（前作で前の問題からヒントが得られる問題があったため）

#### 選択肢補完ルール

- 空文字列 `""`: タレント名からランダム補完（世代フィルタリングあり）
- `[セット名]` 形式: answerSets から該当セットのメンバーをランダム選出
- 世代フィルタ: generation=0 は全タレント、1/2 は該当世代のみ

## 4. タイムアタックモード

### 導線・解放
- タイトル画面に独立ボタンとして配置（解放後に出現）
- 解放条件: 全バッジスロットが最大レベル（2期生知識クイズはブロンズ、他はゴールド）
- 通常モードでは排除した1期生+2期生混合が、タイムアタック限定で登場する

### ゲーム仕様
- **問題数**: 固定100問
- **出題構成**: 全問題タイプ（通常モードの問題タイプ + タイムアタック専用）を決まった順番で決まった数ずつ出題。レベルを段階的に上げていく
- **難易度**: 固定（100問の中にLv.1〜Lv.3が含まれる）
- **制限時間**: なし（全100問クリアまでの合計タイムを計測）
- **不正解ペナルティ**: 問題タイプに応じたペナルティ秒数をクリアタイムに加算。ペナルティの内訳は結果画面に表示しない
  - 名前を作ろう: +15秒、ぼかし当て/スポットライト: +10秒、その他: +5秒
  - 「名前はどこ？」の誤クリックはペナルティ対象外（不正解扱いしない）
- **クリアタイム**: 実プレイ時間（タイマー一時停止中を除く）+ ペナルティ秒数
- **回答後の進行**: 回答後タイマー一時停止 → 解説確認可能 →「次へ」ボタンでタイマー再開
  - 配信者が問題について話す時間を確保できる
  - RTA的に遊びたい人は即座に「次へ」を押せばよい

### 画面フロー
- タイトル画面の「タイムアタック」ボタン → 確認ダイアログ → 「スタート」ボタン → 3秒カウントダウン（リングアニメーション） → クイズ開始
- **中断**: 「やめる」ボタンで確認ダイアログ表示（ダイアログ表示中もタイマーは進む）。OKならタイトルへ戻る
- **結果画面**: クリアタイム、自己ベスト（更新時は更新表示）、X共有ボタン、トップに戻るボタン

### タイマー表示
- ヘッダー内の進捗リング横に表示
- フォーマット: `MM:SS.s`（1/10秒単位）
- 回答後（一時停止中）は最後の値で固定表示
