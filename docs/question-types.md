## 1. 顔名前当ての進化

| 世代 | 方式 |
|------|------|
| 前々作 | 顔当て・名前当てが別々のゲームモード |
| 前作 | 「顔名前当て」として顔当て・名前当てがランダムに混合出題。知識クイズは別モード |
| **2nd** | 顔当て・名前当てに加え新しい出題パターンを追加。知識クイズも問題タイプの一つとして統合 |

- 各問題タイプごとに難易度を設ける（段階数は問題タイプにより異なる。必ずしも3段階でなくてよい）
- 問題タイプの追加が容易なプラグイン的アーキテクチャを採用する（詳細は [architecture.md](architecture.md)）
- 設定画面で問題タイプの選択（チェックボックスでON/OFF）ができるようにする
  - 初期状態では従来の顔名前当てのみ有効
  - 特定条件（アチーブメント達成等）を満たすと他の問題タイプが解放される
- **問題タイプは今後追加する可能性がある**

---

## 2. 出題方式

- 設定画面で選んだ難易度に応じて、解放済みの各問題タイプが**規定数ずつ**出題される
- 出題範囲の全タレントが出題対象（問題数の選択肢は不要）
- 問題タイプの出題順や各タイプの問題数はプログラム側で制御する（詳細は実装時に調整）

## 3. 問題タイプ一覧・難易度別仕様

※ タイプ名はすべて仮称

| # | タイプ名（仮） | 概要 | ★☆☆ | ★★☆ | ★★★ | 備考 |
|---|--------------|------|------|------|------|------|
| 1 | 顔当て | 名前→顔画像の4択 | 選択肢は出題範囲内でランダム | 選択肢は出題範囲内で似た髪色/髪型優先 | 選択肢画像がシルエット。選択肢は全タレントで似た髪型優先 | 前作の顔名前当て（顔→名前）を独立化 |
| 2 | 名前当て | 顔画像→名前の4択 | 選択肢は出題範囲内でランダム | 選択肢は出題範囲内で似た髪色/髪型優先 | 出題画像がシルエット。選択肢は出題範囲内で似た髪型優先 | 前作の顔名前当て（名前→顔）を独立化。全難易度で選択肢は出題範囲内に限定 |
| 3 | 名前を作ろう | タレント名の文字+ダミー文字から順番にクリックして名前を完成。クリアボタンでクリア可能。決定ボタンで回答提出 | 正しい苗字・正しい名前・他人の苗字×2・他人の名前×2の中から正しい苗字・正しい名前を選択する | 正しい氏名の文字（1文字ずつ）・他人の氏名の文字（1文字ずつ）の合計15文字から1文字ずつ順番に選んで名前を作って回答 | 基本的に★★☆と同じ。紛らわしい文字が追加され（日/火/陽/飛/緋/灯など）、合計文字数を35に | |
| 4 | 知識クイズ | 4択の知識問題。questions.jsonから出題 | difficulty 0（基本問題相当） | difficulty 1〜2 | difficulty 3〜5 | 前作の基本問題+深堀りを統合。データ設計の詳細は [data-design.md](data-design.md) 参照 |
| 5 | プロフィール→顔当て | 名前を含まないプロフィール情報→名前付き顔画像の4択 | - | - | 人によって難易度が違うと思われる。便宜上★★★とする | **タイムアタック専用**。前々作の難易度「鬼」に相当 |
| 6 | 名前はどこ？ | 文字グリッドの中から指定タレントの名前を見つけてクリック | - | - | グリッドサイズ30×15 | **タイムアタック専用** |
| 7 | スポットライト | タレント画像のスポットライトが当たった部分だけ見える状態で名前を当てる。スポットライトはランダムに動く | - | スポットライトは3つ | スポットライトは1つでやや高速に動く | **タイムアタック専用** |
| 8 | ぼかし | 加工画像がだんだん正常になっていく過程で名前を当てる | - | ぼかし→鮮明 | ぼかし→鮮明に加えて同時にモノクロ→色付きの変化もある | **タイムアタック専用** |

- 上記すべてを初期リリースに含めるかは優先度を見て判断する
- 初期状態では問題タイプ1・2（顔当て・名前当て）のみ有効。他は特定条件で解放
- 問題タイプは今後追加する可能性がある

### 通常モードとタイムアタックの対応

- 問題タイプ1〜3（顔当て・名前当て・名前を作ろう）: 通常モードの顔名前当て + タイムアタック
- 問題タイプ4（知識クイズ）: 通常モードでは独立モードとして出題。タイムアタックでは他の問題タイプと混合出題
- 問題タイプ5〜8（プロフィール→顔当て・名前はどこ？・スポットライト・ぼかし）: **タイムアタック専用**

### 知識クイズの補足

- 前作の「基本問題」（difficulty 0）と「深堀り問題」（difficulty 1〜5）を難易度で一本化
- 2期生の知識クイズについて:
  - 2期生固有の深堀り問題は十分な問題数が用意できるまで保留
  - ただし、パレ学や寮自体に関する問題（タレント個人に依存しない問題）は前作からそのまま流用可能
  - これにより2期生の知識クイズも初期リリースからある程度の問題数で提供できる

## 4. タイムアタックモード

### 導線・解放
- タイトル画面に独立ボタンとして配置（解放後に出現）
- 解放条件: 全バッジスロットをゴールドにする
- 通常モードでは排除した1期生+2期生混合が、タイムアタック限定で登場する

### ゲーム仕様
- **問題数**: 固定100問
- **出題構成**: 全問題タイプ（通常モードの問題タイプ + タイムアタック専用）を決まった順番で決まった数ずつ出題。レベルを段階的に上げていく
- **難易度**: 固定（100問の中にLv.1〜Lv.3が含まれる）
- **制限時間**: なし（全100問クリアまでの合計タイムを計測）
- **不正解ペナルティ**: タイムに秒数加算（例: +5秒）
- **回答後の進行**: 回答後タイマー一時停止 → 解説確認可能 →「次へ」ボタンでタイマー再開
  - 配信者が問題について話す時間を確保できる
  - RTA的に遊びたい人は即座に「次へ」を押せばよい

### 出題構成イメージ（問題タイプ数や配分は確定後に調整）
```
Lv.1 (★☆☆相当): 各タイプ × 数問 = 約33問
Lv.2 (★★☆相当): 各タイプ × 数問 = 約33問
Lv.3 (★★★相当): 各タイプ × 数問 = 約34問
合計: 100問
```

### ランキング・記録
- 自己ベストタイムをLocalStorageに保存
- Xに結果を投稿する導線を用意（ハッシュタグで擬似ランキング）
- 余裕があればGoogle Sheets連携等で簡易ランキングを検討
