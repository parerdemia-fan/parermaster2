# 隠し要素アイデア集

世代マスター称号で解放される隠し要素の候補。タレントに台詞を喋らせない前提。

---

1. **タレントカード図鑑** — 全問正解した問題の正答タレントがカードとして収集される。カードにはミニトリビア（誕生日、身長等のプロフィール情報）が載る。コンプリート要素
2. **くろぴの裏メモ** — AI開発者くろぴが各タレントについて調べたときの所感・メモを読める（「この子の名前の漢字が難しくて覚えるのに苦労しました」等）。開発日誌の個人版
3. **プレイ統計ダッシュボード** — 自分のプレイ履歴を可視化。最も間違えたタレント、最も早く正解したタレント、寮別正答率、累計プレイ時間など
4. **隠しクイズモード「逆クイズ」** — 通常とは逆のクイズ。例：プロフィール情報が表示され「これは誰のプロフィールでしょう？」、または2人のタレントが表示され「どちらが年上？」「どちらの身長が高い？」など
5. **壁紙・テーマ変更** — ゲーム背景やUIカラーテーマを変更できる。寮カラーテーマ（バゥ/ミュゥ/クゥ/ウィニー）や季節テーマなど
6. **タレント相性診断** — プレイヤーの回答傾向（どの寮のタレントをよく覚えているか等）から「あなたに合う寮」「あなたが推しているかもしれない寮生」を診断。Xシェア対応
7. **寮別ミニゲーム** — クイズとは別の小さな遊び。例：タレントの名前でしりとり、名前の画数当て、タレント神経衰弱（顔と名前のペアを当てる）など
8. **デイリーチャレンジ** — 日替わりで1問だけ出題される特別な高難易度問題。連続正解日数を記録。カレンダーUIで達成状況を表示
9. **タレントシャッフル壁紙** — アクセスするたびにランダムなタレントの立ち絵が表示されるスクリーン。推しの画像をダウンロードできる機能付き（画像は既存アセットを流用）
10. **くろぴの部屋** — AI開発者くろぴとの簡易チャット風画面。定型応答で開発裏話やゲームのヒントを語る。前々作のAIストーリー要素を台詞問題なく再現できる形式
