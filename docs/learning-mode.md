# おぼえよう

寮生の顔と名前をまだ覚えていない人向けの学習モード。3人ずつ覚えて確認するサイクルを繰り返す。

---

## 背景

パレデミア学園を最近知った人がいきなり顔名前当てクイズに挑戦するのは無理ゲーである。まず顔と名前を覚えるための導線が必要。

## 基本フロー

### 0. 初期化

出題範囲（2期生全員/1期生全員/寮別）のタレントで**出題リスト**を作成する。**覚えたリスト**は空。

### 1. 暗記フェーズ

- 出題リストの先頭から3人を取り出す
- 3人の**顔画像と名前**を並べて表示
- プレイヤーが「覚えた！」ボタンを押すとテストフェーズへ

### 2. テストフェーズ

- 3人のうち1人の**名前だけ**を表示し、3択で顔画像を出題（顔当て形式）
- **正解** → そのタレントを覚えたリストに入れる
- **不正解** → 正解を表示した上で、そのタレントを出題リストの**末尾に戻す**
- 3人全員をテストしたら、次の暗記フェーズ（ステップ1）へ

### 3. 繰り返し〜終了

- 出題リストが空になるまでステップ1→2を繰り返す
- 出題リストの残りが1〜2人の場合、覚えたリストからランダムに補充して3人にする
- 全員が覚えたリストに入ったらクリア

## UI

### 暗記フェーズ

- 3人の顔画像＋名前を横並びで表示
- カードがめくられるアニメーション演出があっても良い（必須ではない）
- 下部に「覚えた！」ボタン

### テストフェーズ

- 上部: 「○○はどれ？」の大きいテキスト
- 下部: 顔画像3枚を横並び
- 正解時: 正解演出（通常クイズと同じ）
- 不正解時: 正解を表示（通常クイズと同じ）

### 進捗表示

- リング型プログレス: 「覚えた: X / Y 人」
- ヘッダーに常時表示

## 出題範囲

通常モードと同じ選択肢:
- 2期生全員
- 1期生全員
- 寮別（バゥ/ミュゥ/クゥ/ウィニー）

## 導線

### 世代別（1期生編・2期生編）

設定画面のゲームモード選択に「おぼえよう」を追加。「顔名前当て」「知識クイズ」の左に配置。

```
── ゲーム ──
● おぼえよう  ○ 顔名前当て  ○ 知識クイズ
```

「おぼえよう」選択時は難易度・問題タイプの設定は非表示（不要なため）。

### 寮別モード

寮選択と難易度の間に「おぼえよう」と「顔名前当て」の切替ボタンを配置。

```
── 寮 ──
○バゥ ●ミュゥ ○クゥ ○ウィニー

── ゲーム ──
○ おぼえよう  ● 顔名前当て

── 難易度 ──
●ふつう ○むずかしい 🔒激ムズ
```

「おぼえよう」選択時は難易度の設定は非表示。

## アシスタントのコメント

通常クイズとはニュアンスが異なるため、quotes.json に「おぼえよう」用の場面を追加する。

### 暗記フェーズ

場面名: `おぼえよう_暗記`

テスト前の励まし・声かけ:
- 「この3人をおぼえてね〜！」
- 「じっくり見てね〜」
- 「準備できたらボタン押してね〜」

### テスト正解時

場面名: `おぼえよう_正解`

覚えていたことを褒める:
- 「すごい！おぼえてる〜！」
- 「ばっちり〜！」
- 「さすが〜！」

### テスト不正解時

場面名: `おぼえよう_不正解`

通常クイズの「残念！」ではなく**励まし寄り**にする:
- 「大丈夫、もう1回出てくるよ〜」
- 「次こそおぼえちゃお〜！」
- 「ゆっくりでいいからね〜」

### クリア時

場面名: `おぼえよう_クリア`

- 「全員おぼえたね！すごい〜！」
- 「これでばっちりだね〜！」
