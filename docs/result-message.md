# 結果画面メッセージ設計

## 概要

結果画面に正解率に応じた名前入りメッセージを表示し、Xシェア文言にも含める。

## メッセージ定義（通常ゲーム）

| 正解率 | メッセージ |
|--------|-----------|
| 100% | 🎉 {player}さん、パーフェクト！ |
| 80%以上 | ✨ {player}さん、すごい！ |
| 60%以上 | 👏 {player}さん、なかなか！ |
| 40%以上 | 💪 {player}さん、がんばりました！ |
| 40%未満 | 📖 {player}さん、もう一回チャレンジ！ |

## 表示位置

- 正解率パネルの下部中央に表示
- バッジ/称号獲得時はアニメーション後に表示

## Xシェア文言

### 通常ゲーム

```
{message}

パレ学マスター 2nd Season
{genLabel} {modeLabel} {diffLabel}
{correctCount}/{total}問正解（{rate}%）
#パレ学マスター #パレデミア学園
```

### タイムアタック

```
{playerName}: {timeMessage}

パレ学マスター 2nd Season タイムアタック
クリアタイム: {time}
#パレ学マスター
```
