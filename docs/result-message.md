# 結果画面メッセージ設計

## 概要

結果画面に正解率に応じた名前入りメッセージを表示し、Xシェア文言にも含める。
パーフェクト時はバッジ/称号情報もシェア文言に含める。

## メッセージ定義（通常ゲーム）

| 正解率 | メッセージ |
|--------|-----------|
| 100% | 🎉🎉🎉 {player}さん、パーフェクト達成！ 🎉🎉🎉 |
| 80%以上 | ✨ すばらしい！{player}さん、もう少しでパーフェクト！ |
| 60%以上 | 👏 いい調子！{player}さん、なかなかの実力！ |
| 40%以上 | 💪 {player}さん、まだまだ伸びしろたっぷり！ |
| 40%未満 | 📖 {player}さん、何度でもチャレンジしよう！ |

## 表示位置

- 正解率パネルの下部中央に表示
- バッジ/称号獲得時はアニメーション後に表示

## Xシェア文言

### 通常ゲーム

```
【パレ学マスター 2nd Season 結果発表】
{genLabel} {modeLabel} {diffLabel}
{パーフェクト時: 🎉🎉🎉パーフェクト達成！🎉🎉🎉 / それ以外: 正解数と正解率}
{resultMessage}
{パーフェクト時のみ: 🏆称号 or 🏅バッジ情報}

👇挑戦はこちら
#パレ学マスター #パレデミア学園
```

#### パーフェクト時のバッジ/称号テキスト

- 称号獲得時: `🏆 パレ学マスター達成！` 等
- バッジ獲得時: `🏅 {slotLabel} {rankLabel}バッジ 獲得！`
- バッジランクアップ時: `🏅 {slotLabel} {rankLabel}バッジ ランクアップ！`
- バッジなし（既に獲得済み）: 表示なし

### タイムアタック

```
{playerName}: {timeMessage}

パレ学マスター 2nd Season タイムアタック
クリアタイム: {time}
#パレ学マスター
```
