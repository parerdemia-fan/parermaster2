# 結果画面メッセージ設計

## 概要

結果画面に正解率に応じた名前入りメッセージを表示し、Xシェア文言にも含める。
パーフェクト時はバッジ/称号情報もシェア文言に含める。

## メッセージ定義（通常ゲーム）

| 正解率 | 1行目 | 2行目 |
|--------|-------|-------|
| 100% | 🎉 パーフェクト！ 🎉 | {player}さん、完璧です！ |
| 80%以上 | ✨ すばらしい！ | {player}さん、あと少し！ |
| 60%以上 | 👏 いい調子！ | {player}さん、いい実力！ |
| 40%以上 | 💪 がんばりました！ | {player}さん、伸びしろあり！ |
| 40%未満 | 📖 チャレンジ！ | {player}さん、次こそは！ |

## 表示位置

- 正解率パネルの下部中央に表示
- バッジ/称号獲得時はアニメーション後に表示

## Xシェア文言

詳細は `docs/x-share-texts.md` を参照。以下は概要。

### 通常ゲーム

```
パレ学マスター 2nd Season
{genLabel} {modeLabel} {diffLabel}
{通常: {correctCount}/{total}問正解（{rate}%） + メッセージ}
{パーフェクト: 🎉 {playerName}さん、パーフェクト達成！ + バッジ情報}

{GAME_URL}
#パレ学マスター #パレデミア学園
```

### タイムアタック

```
パレ学マスター 2nd Season タイムアタック
⏱ {time} {自己ベスト更新時: 🎊自己ベスト更新！}
{playerName}: {timeMessage}

{GAME_URL}
#パレ学マスター #パレデミア学園
```
