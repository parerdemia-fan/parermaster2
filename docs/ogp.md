# OGP画像生成画面

## 概要

OGP画像（1200×630px）をスクリーンショットで生成するための専用画面。デバッグ画面からのみ遷移可能。

## サイズ

- **出力サイズ**: 1200 × 630px
- **アスペクト比**: 約 1.9:1
- **画面実装**: GameContainer 外に独立した固定サイズの div で実装（4:3制約を受けない）

## 構成要素

### 1. 背景
- `bg_title.png` を全面に `object-fit: cover` で配置
- やや暗めのオーバーレイ（`rgba(0,0,0,0.2)` 程度）をかけて文字の視認性を確保

### 2. メインロゴ
- `logo_title_landscape.png` を中央上部に大きく配置
- 高さ: 画面の40%程度

### 3. サブテキスト（ロゴ下）
- 「パレデミア学園の寮生クイズゲーム」
- 白文字、やや控えめなサイズ

### 4. 寮エンブレム
- 4寮（`emblem_wa/me/co/wh`）のアイコンを横並び
- 小さめに配置してゲームの多様性を示す

### 5. キャッチコピー（下部）
- 「顔当て・名前当て・知識クイズ…多彩な問題で遊ぼう！」
- 白文字、テキストシャドウで視認性確保

## 実装方針

- `src/app/routes/OgpScreen.tsx` として実装
- 固定サイズ `1200×630px` の div 内にレイアウト
- デバッグ画面に「OGP画像」ボタンを追加して遷移
- `settingsStore` の `screen` に `'ogp'` を追加
- スクリーンショット撮影はブラウザの DevTools またはスクリプトで行う

## スクリーンショット手順

1. デバッグ画面から「OGP画像」ボタンをクリック
2. ブラウザの画面サイズを 1200×630px に設定（DevTools）
3. スクリーンショットを撮影
4. `public/ogp.png` として保存
